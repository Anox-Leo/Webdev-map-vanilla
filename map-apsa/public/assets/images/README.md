# Dossier d'images

Ce dossier contient les images utilisées pour les parcours dans l'application.

Pour que les parcours s'affichent correctement, veuillez ajouter les images suivantes :

- `trail-waterfall.jpg` - Image pour le Sentier des Cascades
- `trail-lake.jpg` - Image pour le Circuit des Lacs
- `trail-vineyard.jpg` - Image pour le Chemin des Vignes
- `trail-mountain.jpg` - Image pour le Sentier des Crêtes
- `trail-old-mill.jpg` - Image pour la Boucle du Vieux Moulin

Si vous n'avez pas d'images spécifiques, vous pouvez utiliser des images génériques de paysages correspondant aux thèmes des parcours.
