# WEBDEV-MAP

To use the full project, you need to install node.js and npm. You can download them from [nodejs.org](https://nodejs.org/).
You can also start ./map-apsa/dist/index.html directly in your browser, but the features will be limited.

## Installation
1. Clone the repository:
    ```bash
    git clone
    cd webdev-map-vanilla/map-apsa
    ```
2. Install the dependencies:
    ```bash
    npm install
    ```
3. Start the development server:
    ```bash
    npm run dev
    ```
## Start WebSocket server
1. Start the server
    ```bash
    cd map-apsa/src/utils/
    node WebSocketServer.js
    ```
